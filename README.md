# Bower + Grunt Project
#### Bower와 Grunt를 활용한 프로젝트

	git clone git@github.com:iduk/grunt-project.git


## *Base install
	yarn install
	yarn global add bower --save
	yarn global add grunt-cli --save
	yarn global node-sass --save
